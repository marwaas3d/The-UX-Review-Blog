## 📌 Overview

The UX Review Blog is a modern blog website that shares insights and opinions about **UX design, technology, and digital culture**.

It focuses on delivering clean UI with bold typography and engaging content.

---

## ✨ Features

- 📰 Latest Articles section  
- 🔥 Trending posts  
- ✍️ Authors section  
- 📂 Multiple categories (Tech, Design, Culture, Opinion, Tutorials)  
- 📧 Newsletter subscription  
- 🎨 Clean and modern UI  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript
